# Spielregeln - Dame

## Figuren:
 - Bauer, Dame

## Farben:
 - Weiß, Schwarz

## Spielfeld:
 - 8x8, weißes Feld rechts unten

## Startaufstellung:
  ```
  x B x B x B x B x B x B x B x
  B x B x B x B x B x B x B x B
  x B x B x B x B x B x B x B x
  ```
x = leer

## Figurzüge:
 - **Bauer:** Immer nur ein Feld diagonal Richtung Gegner / er kann nur eine Figur schlagen, die diagonal rechts oder links von ihm steht und das Feld dahinter frei ist / erreicht er die gegnerische Endlinie wird er zur Dame
 - **Dame:** Zieht und schlägt wie der Bauer + rückwärts

## Situationen:
 - **Kettenschlag**: Hat eine Figur einen Gegner geschlagen und könnte weiterschlagen, kann das soweit fortsetzen wie möglich - Erreicht er die Grundlinie und wird zur Dame, geht kein weiterer Kettenschlag mehr
 - **Matt:** Wenn eine Farbe nicht mehr ziehen kann oder keine Figuren mehr hat
